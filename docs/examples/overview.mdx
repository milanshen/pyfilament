Each example is a self-contained script you can run from the repository root. They build up from a single in-process task to a distributed pipeline, so you can pick the one that matches what you are trying to learn.

## Running the examples

Every example runs as a module from the repository root:

```bash
python -m examples.hello_world
```

The weather agent and web analyst call hosted models, so they need an API key in your environment:

<CodeGroup>
  ```bash Anthropic
  export ANTHROPIC_API_KEY=sk-ant-...
  ```

  ```bash OpenAI
  export OPENAI_API_KEY=sk-...
  ```
</CodeGroup>

The web analyst also needs a running Redis instance for its queue. See [Queues](/distributed/queues) for setup details.
